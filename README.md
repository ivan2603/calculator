# Project Title

Це проект для розрахунку вартості поїздок на різних транспортних засобах за допомогою PHP.

## Встановлення

1. Клонуйте репозиторій:
    ```bash
    git clone <url>
    ```
2. Перейдіть у директорію проекту:
    ```bash
    cd project-root
    ```
3. Встановіть залежності за допомогою Composer:
    ```bash
    composer install
    ```
